## Overview

`Tiempo` is the weather information activity that displays current weather conditions for Huelva, Spain. It integrates with the OpenWeatherMap API using Android Volley for network requests and presents weather data including temperature, humidity, wind speed, and atmospheric conditions.

**Package:** `com.example.huelvapedia.ApiTiempo`

**Extends:** `AppCompatActivity`

## Class Purpose

- Fetches real-time weather data for Huelva from OpenWeatherMap API
- Displays temperature, feels-like temperature, humidity, pressure, wind speed, and cloud coverage
- Translates weather descriptions from English to Spanish
- Shows weather-appropriate icons based on conditions
- Formats temperature data for user-friendly display

## API Integration

### Request Construction

The activity constructs a complete API URL:

```java
String urlCompleta = URL_BASE + "?q=Huelva,ES&appid=" + API_KEY;
// Results in: http://api.openweathermap.org/data/2.5/weather?q=Huelva,ES&appid=c0d138083fb0634d6d5cd36adafc9a77
```

### Volley StringRequest

```java
StringRequest stringRequest = new StringRequest(Request.Method.POST, urlCompleta,
    response -> {
        try
        {
            JSONObject jsonResponse = new JSONObject(response);
            JSONArray jsonArray = jsonResponse.getJSONArray("weather");
            JSONObject jsonObjectWeather = jsonArray.getJSONObject(0);
            String descripcion = jsonObjectWeather.getString("description");
            JSONObject jsonObjectMain = jsonResponse.getJSONObject("main");
            double temp = jsonObjectMain.getDouble("temp") - 273.15;
            double sensacion = jsonObjectMain.getDouble("feels_like") - 273.15;
            float presion = jsonObjectMain.getInt("pressure");
            int humedad = jsonObjectMain.getInt("humidity");
            JSONObject jsonObjectWind = jsonResponse.getJSONObject("wind");
            String viento = jsonObjectWind.getString("speed");
            JSONObject jsonObjectClouds = jsonResponse.getJSONObject("clouds");
            String nubes = jsonObjectClouds.getString("all");

            traducirDescripcion(descripcion);

            tvTemperatura.setText(formatoDecimal.format(temp) + "ºC");
            tvSensacion.setText("Sensacion térmica: " + formatoDecimal.format(sensacion) + "ºC");
            tvHumedad.setText("Humedad: " + humedad + "%");
            tvNubes.setText("Cantidad de nubes: " + nubes + "%");
            tvViento.setText("Velocidad del viento: " + viento + "m/s (metros por segundo)");
            tvPresion.setText("Presión atmosférica: " + presion + " hPa");
        }
        catch (JSONException e)
        {
            e.printStackTrace();
        }
    },
    error -> Toast.makeText(getApplicationContext(), error.toString().trim(), Toast.LENGTH_SHORT).show()
);

RequestQueue requestQueue = Volley.newRequestQueue(getApplicationContext());
requestQueue.add(stringRequest);
```

### JSON Response Parsing

The activity parses the following data from OpenWeatherMap API response:

| JSON Path | Data | Conversion |
|-----------|------|------------|
| `weather[0].description` | Weather description | Translated via `traducirDescripcion()` |
| `main.temp` | Temperature in Kelvin | Converted to Celsius (K - 273.15) |
| `main.feels_like` | Feels-like temperature | Converted to Celsius (K - 273.15) |
| `main.pressure` | Atmospheric pressure | Used directly (hPa) |
| `main.humidity` | Humidity percentage | Used directly (%) |
| `wind.speed` | Wind speed | Used directly (m/s) |
| `clouds.all` | Cloud coverage | Used directly (%) |

## Data Formatting

### Temperature Conversion

OpenWeatherMap returns temperatures in Kelvin. The activity converts to Celsius:

```java
double temp = jsonObjectMain.getDouble("temp") - 273.15;
double sensacion = jsonObjectMain.getDouble("feels_like") - 273.15;
```

### Decimal Formatting

Temperatures are formatted to one decimal place:

```java
private final DecimalFormat formatoDecimal = new DecimalFormat("#.#");

// Usage:
tvTemperatura.setText(formatoDecimal.format(temp) + "ºC");
// Example output: "22.5ºC"
```
