## About HuelvaPedia

HuelvaPedia is an informative Android application designed to showcase the province and city of Huelva, Spain. Built with Java and powered by Firebase Realtime Database, it provides users with a rich, categorized exploration of Huelva's history, gastronomy, landmarks, and more.

## Key Features

<AccordionGroup>
  <Accordion title="Cloud-Based Architecture" icon="database">
    HuelvaPedia leverages Firebase Realtime Database to store and retrieve all encyclopedic content. This cloud-based approach ensures:
    - Data is always up-to-date without requiring app updates
    - Minimal local storage footprint
    - Scalable content management through Firebase Console
  </Accordion>

  <Accordion title="Categorized Browsing" icon="list">
    Navigate through multiple categories including:
    - Historical landmarks and monuments
    - Local gastronomy and traditional dishes
    - Points of interest and tourist attractions
    - Cultural events and traditions
  </Accordion>

  <Accordion title="Weather Information" icon="temperature-half">
    Stay informed with real-time weather data:
    - Current temperature and conditions
    - Thermal sensation
    - Humidity and atmospheric pressure
    - Wind speed and cloud coverage
  </Accordion>

  <Accordion title="Interactive Actions" icon="hand-pointer">
    Engage with content through direct actions:
    - Open locations in maps (Google Maps, etc.)
    - Initiate phone calls to establishments
    - Visit related websites and resources
  </Accordion>
</AccordionGroup>

## Tech Stack

HuelvaPedia is built with modern Android development technologies:

- **Language:** Java
- **Minimum SDK:** API 27 (Android 8.1 - Oreo)
- **Target SDK:** API 30 (Android 11)
- **Database:** Firebase Realtime Database
- **Network:** Android Volley for API requests
- **Image Loading:** Picasso library
- **UI:** AndroidX, Material Design, ConstraintLayout, CardView

## Requirements

<Note>
  **Internet Connection Required:** HuelvaPedia requires an active internet connection to function, as all content is hosted in the cloud via Firebase.
</Note>

- Android Studio (Electric Eel or newer recommended)
- Android SDK (API 30+)
- Java JDK 11 or higher
- Minimum Android Device: Android 8.1 (Oreo)
- Firebase project with Realtime Database configured
